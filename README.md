# Analog Computing for Signal Processing and Communications - Part I: Computing With Microwave Networks

This code package is related to the paper:

M. Nerini and B. Clerckx, "[Analog Computing for Signal Processing and Communications - Part I: Computing With Microwave Networks](https://ieeexplore.ieee.org/document/11281875)," IEEE Trans. Signal Process., 2025.

## Content of Code Package

The file `main_Figi.m` reproduces Fig. *i* in the paper, for *i* = 5, 6, 7.
